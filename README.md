## Software Testing Laboratory with Mini Project
This repository consists of Software testing programs coded in Python for various problem statements
These include
- Commission problem for Boundary Value Testing
- NextDate function for Equivalence Class Testing
- Commission problem for Decision Table Testing
- Triangle problem for Boundary value, Equivalence class and Decision table testing
- Commission problem for Data Flow Testing
- Binary Search Tree

## Web Automation Testing for Max Fashion
